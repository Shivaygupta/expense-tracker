for running this project you need few setups in your system:
        1.Any working IDE(Intellij would be best)
        2.Java 1.8
        3.Maven
        4.Mysql
        5.PostMan for hitting API's

Setup all these in your system then you can run this and test working of All API's.
Change username and password of mysql in application.properties according to your database user.
